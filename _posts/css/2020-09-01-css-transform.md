# transform 변환 속성

| 속성 | 의미 |
|:---|---:|
| transform-origin |  요소 변환의 기준점을 설정 |
| transform-style |  3D 변환 요소의 자식 요소도 3D 변환을 사용할지 설정 |
| perspective | 하위 요소를 관찰하는 원근 거리를 설정 |
| perspective-origin |  원근 거리의 기준점을 설정 |
| perspective-visibility |  3D 변환으로 회전된 요소의 뒷면 숨김을 설정 |
