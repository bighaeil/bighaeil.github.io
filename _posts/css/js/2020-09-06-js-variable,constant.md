# variable

어떤 값을 어딘가(메모리)에 보관할때 사용

이 어딘가의 값이 변할 수 있거나 계속 변하는 경우에 사용

ex)
```js
var num = 10;
```

# constant

변하지 않는 수 - 상수

nameing 은 항상 대문자를 사용하고 띄어쓰기에 언더바(_)를 사용 한다.

ex)
```js
const MY_NAME = "NAME";
```

> ECMA 6 부터는 let, const 도 존재하지만
> 지금은 ECMA 5 기준으로 공부하니 var 만 사용하겠다.
