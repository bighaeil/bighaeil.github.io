# Function

함수 : 특정 코드를 하나의 결과로 처리  

> 어떤 입력을 주었을 때 어떤 연산 후 어떤 결과를 얻을 수 있는 코드

```js
function add(a, b) {
  let result = a + b;
  return result;
}
```

이때  

어떤 입력(`a`, `b`)을 주었을 때

어떤 연산(`a + b`) 후

어떤 결과(`return result`)를 얻을 수 있는 코드
