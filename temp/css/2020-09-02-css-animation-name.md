# animation-name

@keyframes 규칙(애니메이션 프레임)의 이름을 지정 - 개별 속성

| 값 | 의미 | 기본값 |
|:---|:---|---:|
| none | 애니메이션을 지정하지 않음 | 기본값 |
| @keyframes 이름 | 이름이 일치하는 @keyframes 규칙의 애니메이션을 적용 |  |


# animation-duration

애니메이션의 지속 시간을 설정 - 개별 속성

| 값 | 의미 | 기본값 |
|:---|:---|---:|
| 시간 | 지속 시간을 설정 | 0s |
